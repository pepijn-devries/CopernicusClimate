# CopernicusClimate: Search Download and Handle Data from Copernicus Climate Data Service

Subset and download data from EU Copernicus Climate Data Service:
<https://cds.climate.copernicus.eu/>. Import information about the
Earth's past, present and future climate from Copernicus into R without
the need of external software.
